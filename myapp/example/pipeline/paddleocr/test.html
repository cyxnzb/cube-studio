PaddleOCR VL 测试文档

这是一个用于测试 HTML 转 PDF 转 OCR 功能的示例文档。

一、功能概述

本系统支持多种文件格式的 OCR 识别，包括：

  • 图片格式：PNG、JPG、JPEG
  • 文档格式：PDF、DOCX、HTML
重要提示：所有文件在处理前都会进行格式验证，只有符合要求的文件才能继续执行 OCR 识别。

二、处理流程

系统会根据文件类型自动选择相应的处理流程：

  1. 文件格式检查（支持：docx, pdf, png, jpg, jpeg, html）
  2. 文件类型分类（图片、PDF、DOCX、HTML）
  3. 格式转换（如需要）
  4. PaddleOCR VL 识别
  5. 生成 Markdown 输出

三、技术特点

  • 使用 LibreOffice 处理 Word 文档
  • 使用 weasyprint 处理 HTML 文件
  • 使用 PaddleOCRVL 进行高精度 OCR 识别
  • 自动清理临时文件，避免磁盘空间浪费
